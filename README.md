# simple-ele-commerce-system

## 概述

数据库大作业，一个简单的电商平台，采用前后端分离的设计。系统模块采用总-分式设计，以角色做区分，包括：

* 登录界面
* 用户登录
  * 商城首页
  * 购物车
  * 订单管理
* 商户登录
  * 商品管理
  * 供货商管理
* 供货商登录
  * 货物管理
  * 客户管理
* 平台登录
  * 总体管理

其中商户-供货商-平台登录采用相同前端界面设计

* 数据表符合规范
* 数据库设置（内存）
* 权限管理
